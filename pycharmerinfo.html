2. Features of Py Charmer

2.1 Core Features

Py Charmer is designed for simplicity, performance, and versatility, leveraging Skulpt for browser-based Python execution. Its key features include:

  • Lightweight and Serverless: Runs entirely in the browser using Skulpt, eliminating the need for server-side processing or external libraries, making it ideal for resource-constrained environments.
  • Real-Time Execution: Executes Python code instantly via a text console or script injection popup, supporting both simple expressions (e.g., 1.5+2.3) and complex scripts.
  • Integrated Calculator Module: Includes a built-in calculator module for arithmetic operations, supporting operators like +, -, *, /, **, and % with robust error handling for invalid inputs or division by zero.
  • Error AI Diagnostics: Features a troubleshooting tool that analyzes Python and JavaScript errors, providing actionable suggestions for debugging syntax or runtime issues.
  • Clipboard Integration: Allows users to copy the calculator module script to the clipboard for easy injection, with fallback manual copying for unsupported browsers.
  • Console Management: Supports commands like clear to reset the console and help to display usage instructions, enhancing user experience.
  • Matrix-Inspired UI: Features a neon green, retro-futuristic interface optimized for readability and low resource consumption, with responsive design for mobile and desktop.

2.2 Technical Specifications

  • Python Version: Supports Python 3.8 via Skulpt 1.2.0, ensuring compatibility with standard Python syntax and libraries like re.
  • Dependencies: Minimal, using only Skulpt and Skulpt-stdlib from CDN (jsDelivr), ensuring fast loading and no local setup.
  • Browser Compatibility: Works on modern browsers (Chrome, Firefox, Safari) with WebAssembly support for efficient execution.
  • Resource Usage: Optimized for low memory and CPU usage, suitable for edge devices and low-power systems.

4. How to Use Py Charmer

4.1 Console-Based Execution

Users can enter Python expressions or commands directly in the text console:

  • Arithmetic Expressions: Type expressions like 1.5+2.3 or 2**3 and press Enter or click "Execute" to run the calculator module. Example output: =3.800000 or =8.
  • Python Scripts: Enter simple scripts like print("Hello, World!") for immediate execution.
  • Help Command: Type help or calculator_help() to display usage instructions for the calculator module.
  • Clear Console: Type clear to reset the console.

4.2 Script Injection

For complex scripts, use the "Inject" popup:

  • Click "Inject" to open the popup.
  • Paste or write a Python script (e.g., the calculator module or custom code).
  • Click "Inject Script" to execute. Example: Injecting the calculator module allows calling calculator("5+3") to output =8.

Example Script Injection:

def greet(name): print(f"Hello, {name}!") greet("WebXOS")

Output: [INFO] Hello, WebXOS!

4.3 Troubleshooting

Click "Troubleshoot" to run Error AI diagnostics, which checks for:

  • Python syntax errors (e.g., missing colons in loops).
  • JavaScript runtime stability.
  • CSS validation for UI integrity.

Example: Entering print(Hello) triggers an error: Python error: NameError: name 'Hello' is not defined. Suggestion: Use quotes for strings, e.g., print('Hello').

4.4 Ejecting the Console

Click "Eject" to clear the console and script input, resetting the environment for new code.
